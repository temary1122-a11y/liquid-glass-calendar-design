## 📋 Контекст проекта

**Тип приложения:** Персональное приложение одного мастера ресниц  
**Назначение:** Автоматизация рутинной работы администратора по бронированию  

**Workflow:**
1. Клиент выбирает дату/время в Mini App
2. Форма отправляется администратору
3. Администратор обсуждает детали лично (тип ресниц, цена)
4. Администратор просит предоплату
5. После предоплаты → запись подтверждается (waitlist → confirmed)

**Ограничения:**
- Один мастер (не экосистема)
- Услуга обсуждается лично, не в приложении
- Портфолио в канале Telegram, не в боте

### 🔴 ПРИОРИТЕТ 1: Критические исправления

#### 1.1 Исправить дублирование BOT_TOKEN в config.py
**Файл:** `config.py`  
**Проблема:** Дублирование на строках 16 и 33  
**Текущий код:**
```python
# Строка 16
BOT_TOKEN: str = os.getenv("BOT_TOKEN", "YOUR_BOT_TOKEN_HERE")

# Строка 18-27 (fallback логика)
if BOT_TOKEN == "YOUR_BOT_TOKEN_HERE":
    try:
        with open('.env', 'r') as f:
            for line in f:
                if line.startswith('BOT_TOKEN='):
                    BOT_TOKEN = line.split('=', 1)[1].strip()
                    break
    except:
        pass

# Строка 33 (дублирование!)
BOT_TOKEN: str = os.getenv("BOT_TOKEN", "").strip()
```

**Решение:**
- Удалить строки 18-27 (сломанная fallback логика)
- Удалить строку 33 (дублирование)
- Оставить только строку 16 с правильным fallback
- Добавить валидацию что токен не пустой

**Ожидаемый результат:**
```python
BOT_TOKEN: str = os.getenv("BOT_TOKEN")
if not BOT_TOKEN:
    raise ValueError("BOT_TOKEN is required in environment variables")
```
