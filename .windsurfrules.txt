## ПРОЕКТНАЯ СТРУКТУРА

# Основные директории:
# - src/ — весь исходный код
# - src/components/ — React компоненты
# - src/config.ts — КОРНЕВОЙ ФАЙЛ КОНФИГУРАЦИЦИИ (все настройки здесь)
# - src/hooks/ — кастомные React хуки
# - src/utils/ — утилиты и вспомогательные функции
# - public/ — статические файлы (background.jpg)

# ============================================================
# КРИТИЧЕСКИЕ ПРАВИЛА — НИКОГДА НЕ НАРУШАТЬ
# ============================================================

1. ВСЕ настройки изменяются ТОЛЬКО в src/config.ts
   - Шаблоны сообщений
   - Контактная информация
   - Ссылки на соцсети
   - Токены и ID
   - Настройки интерфейса

2. НЕ изменять структуру CSS классов без необходимости
   - liquid-glass-calendar — основной стиль календаря
   - liquid-glass-admin — стиль админ-панели
   - liquid-glass-tab — стиль табов
   - warm-bg — теплый фон градиент

3. ВСЕ импорты должны быть из src/config.ts для настроек
   - Не использовать import.meta.env напрямую в компонентах
   - Использовать SOCIAL_LINKS, CONTACT_INFO, MESSAGE_TEMPLATES из config

4. НЕ создавать лишних файлов без необходимости
   - Всегда проверять можно ли использовать существующий файл
   - Предпочитать расширение существующего файла созданию нового

5. При работе с формами и шаблонами сообщений:
   - Использовать MESSAGE_TEMPLATES из config.ts
   - НЕ хардкодить тексты сообщений в компонентах

# ============================================================
# КОД СТИЛЬ
# ============================================================

1. TypeScript строгий режим:
   - Всегда указывать типы
   - Использовать interface для объектов
   - Избегать any

2. React компоненты:
   - Использовать функциональные компоненты
   - Хуки в начале компонента
   - Props с типизацией

3. Стилизация:
   - Использовать Tailwind CSS
   - Классы в camelCase или kebab-case согласно существующему коду
   - Сохранять liquid glass эстетику

4. Именование:
   - Компоненты: PascalCase
   - Функции: camelCase
   - Константы: UPPER_SNAKE_CASE
   - Файлы: PascalCase для компонентов, camelCase для утилит
